# 可轉債／事件策略 — 探索分析報告

> 全部為**離線回測**(讀 `twstock.db`,不再吃 FinMind 額度)。分析腳本在 `scratch/`。
> 報酬皆為**絕對報酬、未扣交易成本/滑價**;要看的是**相對「基準」的差**,不是絕對數字。

## 0. 資料與方法

- **價格**:`kline`(310 檔現有 CB 股 + 事件股,4 年日線)。
- **籌碼**:`institutional`(投信/外資/自營淨買賣超)、`margin`(融資/融券餘額)。逐檔抓,已覆蓋全部 CB 股。
- **可轉債**:`cb`(TPEx 快照,轉換價=**發行時、未還原調整**;僅含**目前掛牌中**)。
- **事件**:`events`(公司債/增資 申報,分 稿本/生效;`filed_at` 為上傳日)。
- **指標**:
  - 固定持有 `T+N` 收盤報酬;
  - **MFE**(進場後 N 日內最高點 ÷ 進場 −1)= 抓到峰值賣掉的**上界**;
  - **MAE**(同窗最低點)= 期間最大回檔/風險面;
  - **噴≥20%** = MFE ≥ 20% 的事件比例(我們主要優化的目標);
  - 一律對照「**基準**」(同一批股票、同窗、全日)。

## 1. CB:單純「靠近轉換價」→ 沒有 edge

±5% 進場事件(~1,900 筆 / 200+ 檔)對照基準:

| 持有期 | 靠近轉換價 平均/勝率 | 基準 平均/勝率 |
|---|---|---|
| T+20 | −0.07% / 42.5% | +0.30% / 43.7% |
| T+60 | −1.08% / 37.2% | +0.14% / 39.9% |

**結論**:靠近轉換價本身略**遜於**基準。「促轉換拉抬」在這個切法下看不到後續超額。

## 2. CB:「靠近轉換價 × 距到期/賣回 <180 天」→ 凸性(賭右尾)

| 桶 | T+60 平均 | 勝率 | T+60 中位 |
|---|---|---|---|
| 近轉換價 × 到期<180d | **+9.43%** | 47.3% | −1.40% |
| 近轉換價 × 到期≥180d | −1.74% | 36.6% | −4.96% |
| 基準 | +0.14% | 39.9% | +0.50% |

**平均高、但中位數負、勝率<50%** → 典型**肥右尾**:多數小賠,偶爾軋空大賺(T+60 p90 +67%、max +153%,剔除前 3 大仍 +5% 平均)。對應「發行人到期前拉股價逼轉股、免還現金」。**只能一籃子小部位賭右尾,不能單壓**。

## 3. 用「月內最高點(MFE)」重算獲利 → 揭露關鍵:噴完會縮回去

| 訊號 | 月內最高 中位 | 噴≥20% | 月內最低 中位/p25 |
|---|---|---|---|
| CB 近轉換價 × 到期<180d | +11.3% | **26.6%** | −7.8% / −13.5% |
| CB 近轉換價 × 到期≥180d | +6.4% | 11.8% | −7.2% / −12.7% |
| CB 基準(轉換窗內) | +6.0% | 12.8% | −6.1% / −10.6% |
| 投信買超(5日) | +8.4% | 18.6% | −5.6% / −10.3% |
| 全日基準 | +6.7% | 15.1% | −5.2% / −10.0% |

同一個 CB 桶,用 **T+20 收盤**算是中位負/勝率<50%,用**月內最高點**卻是中位 +11%、約 **1/4 機率噴 ≥20%**。
→ **行情有,但 T+20 前多半縮回去**。**這策略必須主動停利(噴出就賣),不能抱到固定天數**。

## 4. 大爆發 CB 的長相(贏家 vs 輸家 profile)

近轉換價×到期<180d 事件,按 T+60 分上/下四分位:

| 特徵 | 贏家(中位+53%) | 輸家(中位−20%) |
|---|---|---|
| 股價相對轉換價 | **−3.2%(仍在下方)** | **+3.6%(已在上方)** |
| 前20日動能 | **+7.3%** | −1.5% |
| 發行額 | 5 億 | 8 億 |
| 距到期 | ~127 日 | ~61 日 |
| 票面利率 | 0% | 0%(全樣本皆零息) |

**能區分的**:① 股價仍**略低於**轉換價(行情還在前面)、② 近期**已轉強**、③ 發行額較小。
**警告**:右尾集中在少數股票(24 贏家僅 11 檔、前 2 名佔 38%)+ 窗口重疊 → 獨立樣本遠少於 n,profile 屬**假說等級**。

## 5. 拉高「噴≥20%」機率的嘗試 + 過度配適檢驗

| 濾網(基於 §4) | n | 噴≥20% |
|---|---|---|
| 基準(近轉換價×到期<180d) | 143 | 26.6% |
| + 近20日動能>0 | 83 | **33.7%** |
| 低於轉換價 & 動能>0 | 52 | 34.6% |
| 再 & 發行額小(≤5億) | 22 | 31.8%(沒用,拿掉) |

**能用的提升**:加「動能>0」把 27% → ~34%(留得住樣本)。
**但時間切分爆雷**:訓練期(<2025-06)僅 31 筆、噴出率 16%;測試期(≥2025-06)112 筆、29.5%。**事件與噴出率高度集中在近一年的 regime**,濾網**樣本外測不出來**(訓練期只 2 筆通過)。
→ 提升**只是 in-sample 提示,未驗證**;最硬的瓶頸是**缺已到期/下市 CB**(快照 API 給不了 → 存活者偏差,且抓更多現有股也補不到早期事件)。

## 6. 投信買超 + 融券上升(軋空假說)→ edge 全來自投信腿

| 訊號 | T+60 平均 | 勝率 | 中位 |
|---|---|---|---|
| 投信買超 + 融券上升(組合) | +7.89% | 53.6% | +1.63% |
| └ 投信買超 單獨 | +7.87% | 53.1% | +1.63% |
| └ 融券上升 單獨 | +6.07% | 51.2% | +0.59% |
| 基準 | +6.21% | 51.2% | +0.50% |

**「融券上升」單獨 ≈ 基準(無訊號)**;疊上去不但沒加分,還小幅稀釋勝率。**軋空假說未獲支持**;真正有用的只有「**跟投信買超**」,且僅溫和 edge(T+20 約 +1pp 超額、勝率 53–54%)。性質**穩定、回檔淺**,可抱可波段。
*(待測:把「融券上升」換成「券資比/回補天數高」,看高短壓是否才有軋空效果。)*

## 7. 稿本/生效 → 3 個月(60 交易日)

| 階段 | n | 收盤中位/勝率 | MFE中位 | 噴≥20% | p90 | MAE中位/p25 |
|---|---|---|---|---|---|---|
| **增資/生效** | 116 | **−9.8% / 32%** | +9.6% | 31.9% | +41% | −19% / −29% |
| 公司債/生效 | 298 | −1.6% / 47% | +13.3% | 36.2% | +48% | −14% / −21% |
| 增資/稿本 | 165 | −2.0% / 44% | +18.1% | 43.0% | +54% | −15% / −22% |
| **公司債/稿本** | 497 | **+0.8% / 53%** | +18.1% | **46.5%** | **+70%** | −11% / −18% |
| 基準 | ~29萬 | −0.2% / 49% | +12.6% | 34.2% | +49% | −10% / −18% |

- **增資生效 → 偏空**(全表最乾淨):3個月中位 −9.8%、勝率 32%、p25 −29%。**現增生效=稀釋/賣壓落地**。可做「迴避/偏空」。
- **公司債生效 ≈ 中性偏弱**,無獨立 edge。
- **公司債/CB 稿本 → 噴出最強**:噴≥20% 46.5%(基準 34%)、p90 +70%。**消息在「申報草案」就反映,生效已太晚**——與 §2–5 的 CB 發現一致。

## 8. 總結 — 目前的定位

| 策略 | 性質 | 一句話 |
|---|---|---|
| 公司債/CB 稿本(可疊近轉換價×快到期) | 凸性/噴出 | **抓題材要在稿本階段**,一籃子賭右尾、**主動停利** |
| 投信買超 | 穩定小 edge | 跟著投信,溫和、回檔淺 |
| 增資生效 | 反指標 | **迴避或偏空**,稀釋落地拖 3 個月 |
| 單純靠近轉換價 / 融券上升 / 公司債生效 | 無 | 無獨立價值 |

**共同限制**:未扣成本;窗口重疊→獨立樣本偏少;universe 偏(視覽選樣);CB 存活者偏差 + 轉換價未調整;噴出率有明顯**近一年 regime**成分,**尚未樣本外驗證**。目前結論為「值得深入」,**未到可直接下單**。

## 9. (a) 結構濾網疊在稿本上 → 負結果

| 公司債/稿本(W=60,n 已含補抓事件股) | n | 噴≥20% |
|---|---|---|
| BASE | 622 | 43.4% |
| + 動能>0 | 335 | 46.3% |
| + 靠近某在外CB轉換價(±5%) | **7** | 28.6% |
| + 靠近CB & 到期<180d | **0** | — |

**「公司債/稿本 × 近轉換價×快到期」疊不起來(n≈0)**:稿本是**新申報的債**,當下通常沒有舊 CB 正好接近轉換價又快到期——兩訊號在 CB 生命週期的**不同階段**。唯一可加的「動能>0」只 +3pp,且**時間切分未驗證**(訓練期 39.2% 無提升、測試期才有)。

## 10. (b) CB 籌碼確認 → 有效(目前最強槓桿)

CB 近轉換價×到期<180d(W=20,有籌碼的子集),依進場前 5 日籌碼分組:

| 分組 | n | 噴≥20% |
|---|---|---|
| 全部(有籌碼) | 135 | 28.1% |
| **投信買超(trust5>0)** | 19 | **52.6%** |
| 非投信買超 | 116 | 24.1% |
| 融券上升 | 60 | 31.7% |
| 融券未升 | 75 | 25.3% |
| 投信買超 & 融券上升 | 12 | **58.3%** |

**投信買超把噴出率 28% → 53%(近乎翻倍)**;融券上升單獨只小幅幫忙。這是**正交訊號**(非從 CB profile 推導),比結構濾網乾淨。**限制**:n=19/12 很小、樣本期偏近期、無法時間切分驗證(受限於樣本量,而樣本量又受限於缺已到期 CB)。

## 10b. 出場法:落袋多少(realized return)

CB 近轉換價×到期<180d 的「投信買超」子集(n=19),不同出場:

| 出場法 | 中位 | 平均 | 勝率 |
|---|---|---|---|
| 月內最高(天花板) | +20.3% | +19.4% | — |
| 抱到 T+20 | +6.1% | +7.7% | 53% |
| 移動停利 15% | −6.4% | +1.8% | 42% |
| **固定停利 +20%(否則T+20)** | **+20.0%** | +7.1% | **58%** |
| 固定停利 +15% | +15.0% | +4.9% | 58% |

- **移動停利全盤皆輸**(每個參數都遜於呆抱 T+20)——股性太震,噴出前的回檔就洗掉部位。
- **固定停利(限價賣)才對**:投信子集 +20% 停利 → 中位落袋 +20%、勝率 58%。代價是封頂(平均不升),但把策略從「賭尾巴」變「可落袋」。
- **廣義桶(143 筆,無投信過濾)相反**:停利封頂反而砍掉肥尾(其正期望全靠 +76%/+150% 那幾根)→ 廣義桶要**讓獲利奔跑**。

→ 廣義 CB 桶 = 讓利奔跑的賭右尾;投信買超子集 = 可掛 +20% 停利的波段。**兩者出場法相反**。

## 11. 最終定位(更新)

- **最有希望的組合**:**CB 近轉換價 × 快到期(<180d) + 投信買超確認** → 月內噴≥20% 約 50%+,一籃子小部位、**主動停利**。屬「值得用小資金實單試水溫」等級,但**尚未樣本外驗證**。
- **增資生效**:反指標,迴避/偏空(3 個月偏弱、稀釋落地)。
- **結構濾網(動能/發行額)**:in-sample 提示為主,時間切分一致地測不出來 → 不採用。
- **共同瓶頸**:缺已到期/下市 CB(快照 API 給不了)→ 樣本小、近一年 regime 成分高,是所有 CB 結論的硬限制。
